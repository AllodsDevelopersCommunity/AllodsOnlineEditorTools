В этот список попадают поля, значения которых меняются с рангом спелла.

Переменные вставляются в текст по клику в списке. Они автоматически
заменятся на соответствующие значения для каждого ранга.

Те числовые значения, которые одинаковы для всех рангов, в этот список
не попадают. Их значения можно писать непосредственно в тексте.
